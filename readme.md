##日期事件选择组件  （完善中）
#日期组件使用
引入css 只是确保基本样式
```
<link rel="stylesheet" href="timeDate.css">
```
页面结构 只需要确保添加上tclass="time-date" id="timeDate" 
```
<input type="text" class="time-date" id="timeDate">
```
引入基本js
```
<script src="timeDate.js"></script>
```

#demo
```
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>日期选择</title>
	<link rel="stylesheet" href="timeDate.css">
</head>
<body>

	<input type="text" class="time-date" id="timeDate">
	<script src="timeDate.js"></script>
	
</body>
</html>
```
